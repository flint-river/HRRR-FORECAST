Presets

Keys — space play/pause · J/K or arrows step · Home/End ends · C compare previous run · F flicker · R re-detect latest run · W wall mode · 1/2/4 panels. MAG stores images under /data/<model>/<YYYYMMDD>/<HH>/<area>/ Browse directory opens the folder for the selected run. Note that listing is disabled on most of these, so a 403 Forbidden there means the folder exists and the path is right, while a MAG-branded 404 means it does not. Publish lag only sets where the search for the newest run begins.
